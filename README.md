# Ijaz Ahamed - Personal Portfolio

Welcome to the source code for my personal portfolio website! This project serves as a showcase of my academic journey, technical skills, projects, and achievements as a Computer Science student and aspiring developer.

## 🚀 Live Demo
[View Live Portfolio](https://ijazAhamed1234.github.io/PortFololio/) *(After deployment is complete)*

## ✨ Features
- **Modern UI/UX**: Designed with a sleek black, violet, and fuchsia theme.
- **Smooth Animations**: Powered by [Framer Motion](https://www.framer.com/motion/) for fluid page transitions and ambient background effects.
- **Fully Responsive**: Optimized for seamless viewing on mobile, tablet, and desktop screens.
- **Interactive Contact Form**: Configured to open your default email client with pre-filled details.
- **Glassmorphism Design**: Clean UI components with frosted glass effects and glowing borders.

## 🛠️ Tech Stack
- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: Framer Motion
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: GitHub Pages

## ⚙️ Running Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/ijazAhamed1234/PortFololio.git
   cd PortFololio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
